Sogang Ribbon: Sogang univ. restaurant archiving project

서강리본(Sogang Ribbon)은 서강대학교 주변의 식당과 공간을 기록하는 로컬 아카이빙 프로젝트입니다. 단순한 맛집 추천을 넘어, 학생들이 자주 머무는 장소의 분위기와 기억을 시각적으로 정리하는 것을 목표로 했습니다.

저는 이 프로젝트에서 비주얼 브랜딩과 콘텐츠 디자인을 중심으로 작업했습니다. 로고, 리본 컬러, 카드뉴스 레이아웃, 사진 톤, 타이포그래피, 굿즈 이미지까지 하나의 시각 언어로 연결해 ‘서강리본’만의 브랜드 무드를 구축했습니다.

프로젝트는 세 가지 리본 카테고리를 기준으로 구성되었습니다.

  • ❤️‍🔥 카디널 리본 – 친구들과 가볍게 방문하기 좋은 친근한 공간
  • ❤️ 레드 리본 – 음식의 완성도와 개성이 뚜렷한 공간
  • 🩷 핑크 리본 – 분위기와 감정적인 경험이 돋보이는 공간

각 콘텐츠는 정보를 단순히 나열하기보다, 장소의 인상과 감각이 먼저 전달되도록 설계했습니다. 사진의 색감, 여백, 텍스트 배치, 카드 흐름을 조정해 사용자가 피드를 넘기면서도 하나의 작은 매거진을 보는 듯한 경험을 느낄 수 있도록 했습니다.

특히 카드뉴스는 공간 소개용 비주얼 에디토리얼에 가깝게 제작했습니다. 식당의 메뉴, 분위기, 방문 포인트를 직관적으로 전달하면서도, 서강리본의 따뜻하고 아카이빙적인 톤이 유지되도록 디자인했습니다.

이 작업을 통해 로컬 콘텐츠도 충분히 하나의 브랜드 경험으로 확장될 수 있다는 점을 확인했습니다. 서강리본은 작은 학생 프로젝트였지만, 시각 시스템을 통해 일상적인 장소를 기억 가능한 콘텐츠로 바꾸는 실험이었습니다.

Reebok AI Advertising Project: “Be Reebok – 일상을 리복답게”

“Be Reebok – 일상을 리복답게”는 REEBOK의 클래식한 브랜드 이미지를 AI 기반 영상 광고로 재해석한 프로젝트입니다. 제품 자체를 직접적으로 설명하기보다, 일상 속 장면과 감정 안에 브랜드가 자연스럽게 스며드는 방식으로 비주얼 콘셉트를 설계했습니다.

프로젝트의 핵심 방향은 “Reebok = Club C 85”라는 인식을 만드는 것이었습니다. Club C 85를 단순한 스니커즈가 아니라, 학창시절·이별·재회·가족·겨울·크리스마스 같은 시간의 장면 속에 함께하는 클래식 아이템으로 표현했습니다.

영상은 촬영 없이 AI 툴을 활용해 제작했습니다. 이미지 생성, 장면 합성, 영상 변환, 음성 및 음원 제작 과정을 통해 브랜드 광고의 시각적 완성도를 실험했습니다. AI를 단순한 보조 도구가 아니라, 광고 비주얼을 구현하는 제작 방식 자체로 활용한 점이 이 프로젝트의 특징입니다.

저는 광고 콘셉트 기획, 브랜드 문제 정의, 스토리라인 구성, 장면별 톤앤무드 디렉팅을 담당했습니다. 각 영상이 하나의 감성적인 흐름을 갖도록 장면 전환, 색감, 인물의 분위기, 제품 노출 방식까지 조정하며 전체적인 비주얼 방향을 설계했습니다.

이 프로젝트는 브랜드 아이덴티티를 영상 언어로 풀어내는 작업이었습니다. 익숙한 제품을 새로운 세대가 다시 감각할 수 있도록, 리복의 클래식함을 일상적이고 영화적인 이미지로 재구성했습니다.

해커톤 UX/UI Design (클릭 시 이미지 확대)

NomaCircle은 디지털 노마드와 원격 근무자를 위한 공간 탐색 및 커뮤니티 플랫폼입니다. 저는 해커톤 프로젝트에서 서비스의 핵심 UX 흐름과 UI 화면 디자인을 중심으로 작업했습니다.

메인 기능인 Finding Place는 사용자가 주변의 카페와 작업 공간을 빠르게 비교하고 선택할 수 있도록 설계했습니다. 평점, 영업시간, 공간 이미지, 위치 정보를 카드형 UI로 정리해 탐색 과정의 피로도를 줄이고, 필요한 정보를 한눈에 파악할 수 있도록 구성했습니다.

지도 화면은 네이버 지도 API를 기반으로 한 인터페이스를 전제로 디자인했습니다. 위치 기반 마커와 장소 카드가 자연스럽게 연결되도록 화면 구조를 잡았고, 사용자가 지도와 리스트 사이를 오가며 공간을 탐색할 수 있도록 흐름을 설계했습니다.

리뷰 작성 화면은 글래스모피즘 스타일의 오버레이 UI로 제작했습니다. 사용자가 기존 화면의 맥락을 잃지 않으면서도 리뷰를 남길 수 있도록, 배경의 공간감은 유지하고 입력 영역은 명확하게 구분했습니다.

커뮤니티 기능은 목적에 따라 세 가지 보드로 나누어 디자인했습니다. 직군 기반의 Position Board, 관심사 기반의 Real Connect, 자유로운 의견 공유를 위한 Anonymous Board를 각각 다른 정보 구조와 인터랙션 방식으로 설계했습니다.

전체 비주얼은 노마드 라이프스타일에서 영감을 받아 딥 네이비와 바이올렛 그라데이션을 중심으로 구성했습니다. 차분하지만 미래적인 무드를 유지하면서, 산세리프 타이포그래피와 간결한 아이콘을 사용해 가독성과 사용성을 함께 고려했습니다.

이 프로젝트는 단순한 앱 화면 제작이 아니라, 물리적 공간 탐색과 디지털 커뮤니티 경험을 하나의 서비스 흐름으로 연결한 UX/UI 작업이었습니다.
